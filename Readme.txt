# -----------------------------------------------------------------------------------------------------------------#
Librerie e comandi per l'installazione
# -----------------------------------------------------------------------------------------------------------------#

************************************************
Framework web usato
pip install Django
************************************************

************************************************
Librerie grafiche aggiuntive
pip install django-crispy-forms
pip install crispy-bootstrap4
************************************************

************************************************
Libreria per la gestione dei permessi con i gruppi
pip install django-braces
************************************************

************************************************
Libreria per la visualizzazione delle immagini caricate dall'utente
pip install Pillow
************************************************



# -----------------------------------------------------------------------------------------------------------------#
Una volta clonata la repo da git, installare le dipendenze e runnare il server locale con
python manage.py runserver
Al primo avvio, decommentare la chiamata alla creazione dei gruppi nel file "urls.py" del modulo
GestionalePalestra
# -----------------------------------------------------------------------------------------------------------------#
